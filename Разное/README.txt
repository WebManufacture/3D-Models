# Summary

This is simple stand for chinese chap drill bits. For me printed from PLA with 0,25mm layer heigh.
Holes are modelled +0,2mm size.
Made for 1.5mm, 2mm, 2.5mm, 3mm, 3.2mm, 3.5mm, 4mm, 4.5mm, 4.8mm, 5mm, 5.5mm, 6mm, 6.5mm drills.
